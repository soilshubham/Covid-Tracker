# Covid-Tracker 

## Details
This website provides you with easy to read global as well as country based daily reports on Covid-19.<br />

Live Demo: <a href="https://soilshubham.github.io/Covid-Tracker/">soilshubham.github.io/Covid-Tracker</a>

![image](https://user-images.githubusercontent.com/54865101/132267871-b9ae195a-0b18-42c6-b3aa-06200b681f63.png)


## Quick start

1.  Make sure that you have Node.js v14 or above and npm v5 or above installed.
2.  Clone this repo using `git clone --depth=1 https://github.com/soilshubham/Covid-Tracker.git <YOUR_PROJECT_NAME>`.
3.  Move to the appropriate directory: `cd <YOUR_PROJECT_NAME>`.<br />
4.  Run `npm install` in order to install all the required dependencies.<br />
5.  Run `npm start` to see the **COVID Tracker** app at `http://localhost:3000`.

Now you're ready to rumble!
